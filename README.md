## 📋 Regler

- `innerHTML` får **inte** användas för att skapa något
  - Endast tillåtet för att ta bort element
- `index.html` får **inte** modifieras
- Inga fler `.ts`-filer får skapas
- Inga förändringar i `tsconfig.json`

---

## 🚀 Innan du börjar

1. Kör `npm install`

2. Skapa en `.gitignore` och lägg till:

   node_modules
